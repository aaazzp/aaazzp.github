我的第一条博客

搭建我的个人博客，参考了以下三篇文章。

https://jingyan.baidu.com/article/d8072ac47aca0fec95cefd2d.html

https://www.jianshu.com/p/189fd945f38f

http://blog.csdn.net/jzooo/article/details/46781805

安装完Git和Node.js后，
在git bash命令行中安装Hexo： npm install -g hexo （此处可能会被墙）

利用淘宝NPM镜像:
$ npm install -g cnpm –registry=https://registry.npm.taobao.org

然后
$ cnpm -g hexo-cli

最后，如果在关联到github后，发现网址打不开，参考第一遍文章即可。
